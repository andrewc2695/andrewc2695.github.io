Intructions:

Use the robot controls on the left of the screen to input directions to the robot. The directions control which way the robot will walk. The time controls how long the robot will wait before executing the next action. If there is no next action the robot will keep walking in the direction of the final action. All inputs must be entered before hitting start!
Example Inputs:
right 1000,
up 500,
right 2000,
down 2000
The robot will walk right for 1 second, then up for .5 seconds, right for 2 seconds, then as down is the last input it will keep walking down forever.
Once you've inputed all the directions you want click start! Becareful not to touch the endges of the screen or you will lose the level!
Hint: Each background box takes around 500 time to move through!
